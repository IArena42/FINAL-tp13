# Classes Bootstrap pour les Tables

| Catégorie            | Classes principales | Utilité |
|----------------------|---------------------|---------|
| **Table de base**    | `.table` | Active le style par défaut des tables Bootstrap |
| **Couleurs**         | `.table-primary`, `.table-success`, `.table-danger`, `.table-warning`, `.table-info`, `.table-light`, `.table-dark` | Applique une couleur de fond aux lignes ou cellules |
| **Stripes**          | `.table-striped` | Ajoute un effet de bandes alternées |
| **Hover**            | `.table-hover` | Met en surbrillance une ligne au survol |
| **Bordures**         | `.table-bordered` | Ajoute des bordures à toutes les cellules |
| **Sans bordures**    | `.table-borderless` | Supprime toutes les bordures |
| **Compacte**         | `.table-sm` | Réduit la taille des cellules pour une table plus compacte |
| **Responsive**       | `.table-responsive` | Rend la table défilable horizontalement sur petits écrans |
| **Alignement**       | `.align-top`, `.align-middle`, `.align-bottom` | Contrôle l’alignement vertical du contenu |
| **Texte**            | `.text-start`, `.text-center`, `.text-end` | Alignement horizontal du texte dans les cellules |
